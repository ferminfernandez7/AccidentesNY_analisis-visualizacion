# AccidentesNY — Análisis y Visualización

## Descripción
Análisis multivariante y visualización interactiva de datos de accidentes de tráfico en Nueva York. Proyecto final de la asignatura *Análisis Multivariante y Visualización de Datos (AMVD)*.

## Contenido
- `Script_analisis-datos.Rmd` — Script RMarkdown con el análisis multivariante completo.
- `Memoria.html` — Informe generado a partir del script con análisis descriptivo, técnicas multivariantes y conclusiones.
- `App/` — Aplicación web interactiva construida con **R Shiny** para la exploración visual de los datos y resultados.
- `data_sampled.csv` — Dataset muestreado de accidentes de tráfico en NYC.

## Tecnologías
`R` · `RMarkdown` · `Shiny` · `ggplot2`
